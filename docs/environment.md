# 部署环境要求

# 服务端

|  运行环境 | 要求版本  |  推荐版本 |
| ------------ | ------------ | ------------ |
|  JDK |  >=1.8 |  1.8 |
| Mysql  |  >=5.7 | 5.7  |
| nginx或 apache  |  无限制 | 1.20.x  |
| Redis  |  >=6.0 | 7.0.0  |

# 前端

|  运行环境 | 要求版本  |  推荐版本 |
| ------------ | ------------ | ------------ |
|  Node.js |  >=14.18.1 |  14.18.1 |
